障害者支援施設の意思決定支援用WebARツール
外部公開用メモ

このフォルダーは、ChatGPTを経由しない完全静的サイトとして公開できます。

推奨公開先
1. Netlify
2. GitHub Pages
3. Cloudflare Pages

公開時の入口ファイル
index.html

WebARで必要なこと
- HTTPSで公開すること
- ブラウザでカメラ許可を有効にすること
- iPad / iPhoneはSafari、AndroidはChromeを推奨

Netlifyで公開する場合
- このフォルダー全体をアップロードします。
- netlify.toml は同梱済みです。
- 公開後に発行されるURLはログイン不要でアクセスできます。

GitHub Pagesで公開する場合
- このフォルダーの中身をリポジトリへ入れます。
- Pagesの公開元をルートに設定します。
- .nojekyll は同梱済みです。

Cloudflare Pagesで公開する場合
- ビルドコマンドは不要です。
- 公開ディレクトリはルート、またはこのフォルダーです。
